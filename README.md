# Heart Embryology — Study Companion

A single-file study app for the **Development of the Heart** embryology lecture. Notes, flashcards, MCQ + true/false quizzes, mistake review, and a quick-revision reference (anomalies, derivatives, mnemonics) — all in one `index.html` you can open in any browser.

## Use it

- **Online:** open the GitHub Pages URL (set up below).
- **Offline:** download `index.html` and double-click to open. All progress is saved in your browser's local storage.

## What's inside

- 8 note sections covering the lecture, with high-yield, cool-fact, and memory-trick callouts
- 300 flashcards with topic filter and "again / got it" recall
- 280 MCQs + 10 true/false questions, with topic and length filters (10 → all)
- A wrong-answer log so you can revisit things you missed
- Reference tables: 14 embryonic → adult derivatives, 8 congenital anomalies, 12 mnemonics
- Light + dark mode, keyboard shortcuts on flashcards (Space flip, ← Again, → Got it)

## Sharing it with a friend (GitHub Pages)

After pushing this repo:

1. On GitHub: **Settings → Pages**
2. Source: **Deploy from a branch**, branch: `main`, folder: `/ (root)`
3. Save. After a minute or two, your app is live at `https://<username>.github.io/<repo-name>/`.
